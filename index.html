A) HTML Structure - top

This template is fully responsive and created with bootstrap5 framework. Template follow the standard html5 structure. It is starts from header aside, and main content. In "main-wrapper" all main content of Template is available. This is based on SCSS. Here is the general structure.

B) SCSS Files and Structure - top

We are using one SCSS file for the template. In the general section of css file I used the codes to reset the generic style of the html. All required styles are included in this file with well commented. According to the comment you can customize the template easily. Here is the basic style sheet structure.

		
	      assets/
		  css/
		    ├── bootstrap.min.css
		    ├── bootstrap-extended.css
			sass/
			├── main.css
		    ├── dark-theme.css
			├── semi-dark.css
			├── bordered-theme.css
			├── responsive.css

C) JavaScript - top

This theme imports five Javascript files.

  1. bootstrap.bundle.min.js
  2. jquery.min.js
  3. perfect-scrollbar.js
  4. metisMenu.min.js
  5. simplebar.min.js
